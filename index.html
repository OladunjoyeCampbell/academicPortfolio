Projects

Capstone Project Portal

This system facilitates final year project topic registration, supervisor approval, and audit logging at Niger State Polytechnic. Built using Flask with a Google Apps Script back-end, it replaces a Google Form solution with a robust web application.

  • Features: Topic uniqueness, supervisor approval workflow, audit trail
  • Technology: Flask Web App (beta); Google Apps Script; Google Sheets API
  • Modules: AvailableTopics; TakenTopics; Approval Queue; Activity Log

Upcoming Teaching Innovations

Planning to develop a series of interactive tutorials and workshops for first‐year programming courses, integrating Scratch and mastery‐based learning approaches. These initiatives aim to enhance student engagement and address impostor phenomenon early in their computing education journey.
